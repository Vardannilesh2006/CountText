SEO Meta Description Character Limit: Pixel Width Rules

Writing description tags for your website pages is a cornerstone of organic Search Engine Optimization. If your tags are too long, Google will truncate them with an ugly ellipse ("..."), causing potential clients to miss vital call-to-actions. Let's look at how Google actually evaluates limits.

Pixels vs Character Limits

Historically, SEO agencies taught writers to limit Meta Titles to 60 characters and Meta Descriptions to 160 characters. However, Google does not count characters. Google's search engine results page (SERP) measures titles and descriptions using **pixel width**.

  • Desktop Meta Title Limit: 600 pixels (roughly 50-60 characters).
  • Desktop Meta Description Limit: 960 pixels (roughly 120-160 characters).
  • Mobile Meta Description Limit: 680 pixels (roughly 100-120 characters).

Why Character Counts Can Lie

Because Google uses the Arial font, letters have variable widths. A capital **"W"** or **"M"** takes up almost three times as many pixels as a lowercase **"i"** or **"l"**. If you write a title with 58 characters containing many capital letters and wide characters, it will exceed 600px and truncate. Conversely, a 65-character title with narrow letters might display completely.

Always use a canvas-based pixel simulator (like the CountText SEO Analyzer) to measure your absolute pixel width rather than just counting characters.
